## Running locally

this repo was build and tested with node v20, start by installing packages with

```bash
# install packages
npm install
# install ts-node globally
npm i ts-node -g
```

You can run solutions with `ts-node` like this

```bash
ts-node ./src/day1.solve.ts
```

You can start tests with

```bash
npm test
```
